## Projects

### 🔋 Battery Brain
A FastAPI REST API that acts as the decision engine for an industrial battery connected to a solar park in Groningen. Integrates with the Open-Meteo API to fetch real-time solar radiation forecasts, processes the data with Pandas, crosses it with energy market prices, and returns a smart action command — `CHARGE`, `DISCHARGE`, or `HOLD`.

[View Project](https://github.com/gabemaldito/battery-brain)

### 🧹 Hygi
A premium cleaning services marketplace combining Airbnb's minimalist aesthetic with a trust-first scheduling model. Built with React Native and Expo, featuring a dual-sided ecosystem with dedicated flows for Clients and Professionals.

[View Project](https://github.com/gabemaldito/higimac)

### 🌧️ Rain Bot
A TypeScript Telegram bot that detects your real-time location and alerts you if rain is expected in the next 2 hours — powered by the Buienradar API and deployed on Railway.

[View Project](https://github.com/gabemaldito/rain-bot)
